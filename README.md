# homegrown-photo-host

Public image-hosting layer for BirdEye photo publishing. BirdEye's Add Media API
pulls images by public URL, so finalized public-posting photos are served here as
raw URLs. Only photos intended for public business profiles (Google/Apple/Bing/
Facebook) live here.

Structure: one folder per facility. Raw URL pattern:
`https://raw.githubusercontent.com/jonathanphilHG/homegrown-photo-host/main/<facility>/<file>.jpg`
